# Miro-Clone Frontend

This directory contains the frontend code for the Miro-clone application, built with React.

## Project Structure

- `src/`
  - `components/`
    - `auth/`: User authentication components (Login, Register).
    - `calendar/`: Components for the calendar feature.
    - `collaboration/`: Components for real-time collaboration (Whiteboard, Document).
    - `core/`: Core components of the application (Home, NotFound).
    - `podcast/`: Components for the podcast feature.
    - `post/`: Components for posts and comments.
    - `room/`: Components for managing and interacting with rooms.
    - `user/`: Components for user profiles and settings.
  - `utils/`: Utility functions, including the API client.
  - `App.tsx`: Main application component with routing.
  - `index.tsx`: Entry point of the React application.
