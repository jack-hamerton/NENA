NENA Direct Messages (DMs) offer a variety of features centered on privacy, security, and rich communication. These functionalities are consistently available in one-on-one conversations on the mobile app, web version.
Core Communication Features
• Messaging: Send text, voice messages, GIFs, stickers, photos, and documents.
• Media Sharing: Easily share photos and videos, including the ability to send full-sized photos.
• Voice and Video Calls: user Stay connected with private voice and video calls, which also include a screen sharing feature during video calls.
• Message Yourself: user Use a personal chat to keep track of notes, reminders, and messages, which can be pinned for quick access.
Advanced Privacy & Security
• End-to-End Encryption (E2EE): All personal messages, calls, photos, and more are automatically secured with E2EE, meaning only user and the recipient can see or listen to them, not even Nena.
• Disappearing Messages: user Control how long messages stick around by setting them to disappear after a specified duration.
• View Once: user Send photos and videos that vanish after they've been opened once, perfect for sensitive moments user don't want saved.
• Chat Lock: user Add an extra layer of protection to specific conversations by password-protecting them and hiding them in a separate, secure folder.
• Silence Unknown Callers: user Automatically screen out spam and unwanted calls from unknown numbers to reduce interruptions.
• Two-Step Verification: user Add a unique PIN for accessing your account, providing an additional security layer.
• Privacy Controls: user Customize who sees your profile photo, "about" information, last seen, online status, and live location.
Convenience & Usability Features
• Cross-Platform Sync: Messages are synchronized in real-time across your phone, nena Web.
• Phone Notifications: Receive new message alerts directly on user phone.
• Automatic Drafts:user Unfinished messages are saved automatically as drafts.
• Nena AI: Optionally use Nena AI for features like message summaries of unread chats, processed privately so no one else can see them
